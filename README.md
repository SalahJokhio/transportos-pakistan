# Transport OS - Enterprise Transportation Operating System

Complete operating system for transportation companies in Pakistan.

## Project Structure

- **backend/** - NestJS microservices backend
- **frontend/** - Next.js web application
- **mobile/** - Flutter mobile app
- **infrastructure/** - DevOps and deployment configs
- **shared/** - Shared types and constants
- **docs/** - Documentation

## Quick Start

See individual README files in each directory for setup instructions.

## Tech Stack

- **Backend**: NestJS, TypeScript, PostgreSQL, Redis
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, ShadCN UI
- **Mobile**: Flutter, Dart
- **Infrastructure**: Docker, Kubernetes, AWS
